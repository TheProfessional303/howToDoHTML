What is a pseudo-class?

A pseudo-class is a selector that tells the class to only do an action when the statement in the pseudo-class is true. For example: :hover. These are the peudo-classes:

  • occurs when the user is hovering over the element.
  • :active - occurs when the user is activating an element (e.g.- clicking on it)
  • :focus - occurs when the user is focusing on an object (e.g.- a form input field)
