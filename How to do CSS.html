        <h3>pseudo-classes</h3>

        <h4>What is a pseudo-class?</h4>

        <p>

            A pseudo-class is a selector that tells the class to only do an action when the statement in the pseudo-class is true. For example: <span class="code">:hover</span>. These are the peudo-classes:

        </p>

        <ul>

            <li class="code">occurs when the user is hovering over the element.

            <li class="code">:active - occurs when the user is activating an element (e.g.- clicking on it)

            <li class="code">:focus - occurs when the user is focusing on an object (e.g.- a form input field)

        </ul>
