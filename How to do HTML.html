How To Do HTML

What is it?

HTML (HyperText Markup Language) is a programming language used to make websites like this one! If you want, you can right click anything and click "inspect" to see the HTML code. You can do this for fonts and if you like a design or something. That was just a tip, by the way.

How to code in HTML

In HTML, there are things called tags. A tag is something that starts with an angle bracket and ends with another angle bracket. In between is some information to tell each tag apart. For example: <html>

First, we will go through the skeleton of HTML.

  1. <!DOCTYPE html>
    This shows we are using a modern version of HTML, not an old, janky version.
  2. <html>
    This signifies the start of the webpage.
  3. <head>
    This is the start of the head, which is where everything about the webpage is stored.
  4. <meta charset="utf-8">
    This tells the browser that the maker of the webpage only wants english, not complicated characters.
  5. <title>new program</title> (programs may have different titles, so it doesn't have to be "new program")
    It shows the title of the webpage.
  6. </head>
    This shows the end of the head.
  7. <body>
    This shows the start of the body of the webpage.
  8. </body>
    This shows the end of the body of the webpage.
  9. </html>
    This shows the end of the program.

Now that we know about the skeleton, let's continue to how to code.

Headings

In HTML, there are things called headings. There are six types of headings, from biggest to smallest:

  • <h1>
  • <h2>
  • <h3>
  • <h4>
  • <h5>
  • <h6>

Here, <h1> is more important than <h6>. Here are some examples:

This is h1.

This is h2.

This is h3.

This is h4.

This is h5.
This is h6.

To make a heading, type <[h1 to h6]>. Then, type in your heading. Finally, type </[h1 to h6]>. For example, <h4>Rabbits!</h4>
If you type this in the body, you will get:

Rabbits!

So, if you want to show a heading, use <[h1 to h6]>[your heading]<[h1 to h6]>!

Lists

Have you ever made a list? Maybe of groceries, best dog faces, or a wishlist? Well, this is what lists are for! they are made like this:

Ordered lists

<ol>
<li>[Your first item]
<li>[Your second item]
<li>[Your third item]
 And so on, until you want to end your list.
</ol>

 For example,

<ol>
<li>Mango
<li>Orange
<li>Banana
</ol>

 And that makes:

  1. Mango
  2. Orange
  3. Banana

Unordered lists

This is just the same, but ul instead of ol. For example, it would look like:

  • Mango
  • Orange
  • Banana

Links

Have you ever seen a link? If you did, where did you see it? A webpage, of course! how, this is how to make one:

<a href="">

Ok, stop there. I know you have few questions, let's answer them.

  1. What does the "a" stand for?
    Answer: The "a" stands for anchor, because you anchor the website.
  2. What is "href"?
    Answer: "href" means hyper refrence, which is just a fancy word for link.

Okay, now type in the link in the quotation marks. For example:

<a href="https://www.w3schools.com/html/">

Well now, there's an error. You need an end to the a tag! Do it like this:

<a href="https://www.w3schools.com/html/"></a>

Now, type in what you want the link to look like between the two a tags. For example:

<a href="https://www.w3schools.com/html/">HTML</a>
